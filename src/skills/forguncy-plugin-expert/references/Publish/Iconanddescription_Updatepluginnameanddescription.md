# 更新插件名称与描述

## Content

插件的名称与描述会在插件管理中显示。
![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80958786/image2023-1-19_15-53-29.png)

通过修改 PluginConfig.json 的 name 和 description 属性可以修改。
![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80958786/image2023-1-19_15-53-41.png)

修改后重新编译工程，重启设计器即可生效。
