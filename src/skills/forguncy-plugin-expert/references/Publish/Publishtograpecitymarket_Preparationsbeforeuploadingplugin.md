# 上传插件前准备

## Content

在葡萄城市场中上传自己开发的插件，首先需要成为葡萄城市场开发者或者葡萄城市场店铺主。
下面列举了三种方式：如何成为店铺主、如何成为个人开发者、如何成为合作伙伴的开发者。任选一种即可。

## 如何登录葡萄城市场

1. 单击网址[https://marketplace.grapecity.com.cn/](https://marketplace.grapecity.com.cn/)，进入葡萄城市场，单击“登录”。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-43-35.png)
2. 出现登录界面，目前我们支持两种登录或注册方式：微信扫码登录、密码登录。
3. 微信扫码登录。使用微信扫一扫会直接登录或者注册。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-44-40.png)
4. 密码登录。
5. 首次登录时需要进行注册操作。
    ![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-45-12.png)![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-45-23.png)
6. 注册成功后，通过输入账号，密码进行登录。
    ![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-45-43.png)

## 如何成为店铺主

1. 用户进入 [https://marketplace.grapecity.com.cn/](https://marketplace.grapecity.com.cn/) 之后单击用户头像，在下拉框中单击“申请入驻”。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-46-19.png)
2. 选择入驻类型为“葡萄城合作伙伴”。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-46-38.png)
3. 进入“入驻须知”页面，填写对应信息，提交信息进行审核。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-46-53.png)
4. 审核通过后，用户可以进入 [https://marketplace.grapecity.com.cn/seller](https://marketplace.grapecity.com.cn/seller) ，登录成功即可表明该用户已经成为店铺主。

## 如何成为个人开发者

1. 用户进入 [https://marketplace.grapecity.com.cn/](https://marketplace.grapecity.com.cn/) 之后单击用户头像，在下拉框中单击“申请入驻”。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-47-15.png)
2. 选择入驻类型为“个人开发者”。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-47-26.png)
3. 进入“入驻须知”页面，填写对应信息，提交信息进行审核。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-47-43.png)
4. 审核通过后，用户可以进入 [https://marketplace.grapecity.com.cn/seller](https://marketplace.grapecity.com.cn/seller) ，登录成功即可表明该用户已经成为个人开发者。

## 如何成为合作伙伴的开发者

葡萄城市场开发者需要归属于某一个合作伙伴商铺，因此用户需要提前联系好店铺主，店铺主添加用户ID，将该用户纳为该店铺的开发者。

1. 用户进入 [https://marketplace.grapecity.com.cn/](https://marketplace.grapecity.com.cn/) 之后单击用户头像，在下拉框中单击“用户中心”，进入用户中心。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-49-17.png)
2. 用户在“我的信息”界面（[https://marketplace.grapecity.com.cn/own_home?menu=UserCenter_MyMessage](https://marketplace.grapecity.com.cn/own_home?menu=UserCenter_MyMessage)），复制ID号发送给店铺主。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-49-38.png)
3. 店铺主拿到用户ID号，登录到 [https://marketplace.grapecity.com.cn/seller](https://marketplace.grapecity.com.cn/seller) ，进入 “开发者管理”界面，进行添加开发者。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-49-53.png)
4. 该用户再次进入 [https://marketplace.grapecity.com.cn/](https://marketplace.grapecity.com.cn/) ，单击用户头像，下拉框中展示文字“卖家中心”即可表明此时用户已成为一名开发者。![](/DOCUMENT_SITE_LINK_PREFIX_HERE/document-site-files/images/8ca07557-62b8-4219-8ddd-357e505dc985/80960489/image2023-2-27_16-50-11.png)
