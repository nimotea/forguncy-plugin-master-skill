# 活字格V10.0.0.0新增特性

## Content

### 单元格插件

1. [字符串属性增强](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addproperty/stringproperty)

* 支持水印
* 支持多行文本
* 支持多语言

2. [公式属性增强](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addproperty/formulaproperty)

* 支持精确监听特定属性公式结果变化
* 支持多语言

3. [列表属性增强](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addproperty/listproperty)

* 支持列默认值
* 支持指定默认列宽
* 支持设置列值不可重复
* 支持内嵌显示

4. [对象属性增强](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addproperty/objectproperty)

* 支持添加对象描述
* 支持自定义校验

5. [数据源属性增强](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addproperty/datasourceproperty)

* 支持精确监听数据源依赖单元变化

6. 设计时能力增强

* [支持使用无头浏览器实现设计时预览](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/celldesignsupport/supportsdesigntimepreview/design-time-preview-in-Chromium)

* [单元格操作支持自定参数校验及动态隐藏](/solutions/huozige/help/docs/plugindevelopment/plugindevelop/developcelltypeplugin/addcellaction/addadvancedcelloperations/advanced-cell-operations-design-time-support)
